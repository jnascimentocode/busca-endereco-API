# busca-endereco-API

## Linguagem
Python versão 3.10.0

## IDE
VsCode com Python

## Bibliotecas utilizadas
requests==2.26.0

## Escopo 

O programa faz conexão via API com o site VIACEP onde o mesmo retorna um JSON para tratamento das informações e retorno do dados do CEP a pesquisar.
